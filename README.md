### ✅ 커밋 종류

> 수정한 종류에 따라 커밋 메시지를 선택

| 메시지명 | 설명                                                     |
| -------- | -------------------------------------------------------- |
| feat     | 새로운 기능 추가 관련                                    |
| fix      | 버그 수정                                                |
| test     | 테스트 코드, 리팩토링 테스트 코드 추가                   |
| refactor | 코드 리팩토링(기능향상)                                  |
| chore    | 빌드 업무 수정, 패키지 매니저 수정                       |
| docs     | 문서 수정(md, git관련 파일, 이미지파일 수정)             |
| style    | 코드 formatting, 세미콜론(;) 누락, 코드 변경이 없는 경우 |
